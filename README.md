# Speed Up

A lightweight browser extension that lets you control video playback speed on any website — YouTube, Netflix, Twitch, Udemy, you name it.

---

## What it does

- A small speed controller appears when you hover over any video
- Click **−** and **+** to slow down or speed up
- Click the number in the middle to reset back to 1×
- Hides automatically when your mouse is idle, just like YouTube's player controls

**Speed range:** 0.10× to 16×

---

## Keyboard shortcuts

Works while your mouse is over the video:

| Key | Action |
|-----|--------|
| `,` | Slow down |
| `.` | Speed up |
| `R` | Reset to 1× |

### Manual install (Developer Mode)
1. Download or clone this repo
2. Open `chrome://extensions` (or `opera://extensions`)
3. Enable **Developer mode** (top right toggle)
4. Click **Load unpacked** and select the project folder
